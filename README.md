## 环境要求
- python 3.12+（低版本可能也行 没试过）
- Windows(mac说不定有字体问题)
依赖见`pyproject.toml`:
- pygame>=2.6.1

## 运行方法
安装好依赖后运行`gobang\main.py`
建议使用uv搭建虚拟环境运行
```bash
uv sync #安装依赖
uv run mian.py #运行游戏
```
